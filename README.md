# MAC-Spoofer
    A Simple python program which spoofs the Physical address.
    
## Installation 

    git clone https://github.com/c0n5n3d/MAC-Spoofer.git
    
## Execution

    1.) Execute main.py file with root privileges.
    
        sudo python3 main.py 
     
    2.) Your Mac-Address is spoofed.
